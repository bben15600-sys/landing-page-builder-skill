# Brand assets

Drop the DJ Kevin Oforia logo here as **`logo.png`** (this exact path: `dj-kevin/public/brand/logo.png`).

Once the file exists, the site header automatically shows it (until then it
falls back to the styled "DJ KEVIN OFORIA" text wordmark).

Recommended:
- `logo.png` — the main logo (transparent background preferred, ~600px wide+).

After adding the file, commit & push — the deployment updates automatically.
